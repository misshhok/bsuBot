после скачивания папки распаковать ее на рабочий стол, открыть терминал и перейти в папку с проектом.
$ cd desktop
$ cd bsu_auth
!!!!!!
знак "$" вводить не надо
!!!!!!
далее нужно ввести команду:
$ source venv/bin/activate 

открыть в редакторе кода файл data.py и ввести входные данные в словаре (между кавычками)

после успешного прохождения всех предыдущих шагов ввести в терминале команду:
$ python3 main.py
